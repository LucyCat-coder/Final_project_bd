# SQL Project: DDL & DML

## Описание

Проект выполнен в формате Jupyter Notebook и состоит из двух разделов:

### DDL

#### Создание базы данных для музыкального магазина. 
* Реализованы таблицы и связи: 
1. Исполнители
2. Альбомы
3. Треки
4. Клиенты
5. Покупки

### DML

Выполнение SQL-запросов к базе данных **Sakila** для анализа клиентов,
фильмов, категорий, аренд и платежей.

## Используемые технологии

#### SQL
#### SQLite
#### Jupyter Notebook

## Как использовать

* Откройте файл `sakila_music_store.ipynb` в Jupyter Notebook
* Выполнитеячейки по порядку.
